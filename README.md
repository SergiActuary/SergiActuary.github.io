# Web personal · Sergi López Vergara

Web personal de presentación como actuario freelance (Solvencia II, IFRS17,
provisiones de vida y análisis de datos).

Sitio estático de un solo archivo (`index.html`, HTML + CSS, sin dependencias),
publicado con **GitHub Pages** en https://sergiactuary.github.io

## Estructura

- `index.html` — la página completa (estilos incluidos, tema claro/oscuro automático).
- `CV-Sergi-Lopez-Vergara.pdf` — CV enlazado desde el botón de descarga.

## Editar

Abre `index.html` en el navegador para previsualizar. Los cambios que se suban a
la rama `main` se publican automáticamente en unos minutos.
