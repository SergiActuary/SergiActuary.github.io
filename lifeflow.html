LifeFlow

Una librería de Python que lleva los cálculos actuariales de carteras de vida del Excel al código: los replica con facilidad, los deja reproducibles y los automatiza. Menos trabajo manual, menos errores, resultados auditables.

¿Te interesa? Hablemos → GitHub · próximamente

El problema que resuelve

Del Excel al código, sin perder el control.

En muchos equipos, el cálculo de una cartera de vida vive en una hoja de Excel: potente para empezar, pero difícil de auditar, de repetir sin errores y de escalar. Cada cierre implica repetir pasos manuales y confiar en fórmulas enredadas que solo una persona entiende.

LifeFlow es un conjunto de funciones preparadas para trasladar ese cálculo a Python de forma directa. Tomas lo que hace el Excel sobre la cartera y lo replicas en código; a partir de ahí, el proceso queda automatizado, versionado y listo para ejecutarse las veces que haga falta con total trazabilidad.

Qué aporta

Ideas principales del proyecto.

📊➡️🐍

Del Excel al código

Replica en Python los cálculos que hoy están en la hoja de cálculo, con una API pensada para el flujo actuarial.

⚙️

Automatización

Una vez trasladado, el cálculo se ejecuta solo: sin pasos manuales repetitivos en cada cierre.

🔁

Reproducible y auditable

Mismos datos, mismo resultado. Todo queda versionado y es fácil de revisar y validar.

🧩

Funciones reutilizables

Bloques preparados para los pasos comunes del cálculo, para no reinventar la rueda en cada cartera.

Idea de uso

Ejemplo ilustrativo — la API final puede cambiar.

# Cargar la cartera desde el mismo Excel de siempre
import lifeflow as lf

cartera = lf.leer_cartera("cartera_vida.xlsx")

# Replicar el cálculo que antes hacía la hoja de cálculo
resultado = lf.calcular_provisiones(cartera, base="IFRS4")

# Automatizado y reproducible: exportar el resultado
resultado.exportar("provisiones_2026.xlsx")

* Fragmento orientativo para mostrar el enfoque. La documentación completa vivirá en el repositorio de LifeFlow cuando esté publicado.
